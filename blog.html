Question-1: What are the different ways to select an element in the DOM?
  • getElementById: Selects an element by its id.
  • getElementsByClassName: Selects elements by their class.
  • getElementsByTagName: Selects elements by their tag (like div, p).
  • querySelector: Selects the first element that matches a CSS selector.
  • querySelectorAll: Selects all elements that match a CSS selector.
Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element

innerText: Gets or sets the visible text inside an element, excluding HTML tags

textContent: Similar to innerText, but it also includes hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach an event listener to a parent element instead of multiple child elements. This allows events on child elements to be handled by the parent, making it more efficient.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and bubbles up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element: Use document.createElement("tag") to create an element;

Add an element: Use parentElement.appendChild() to add the created element.

Remove an element: Use parentElement.removeChild() to remove an element.
